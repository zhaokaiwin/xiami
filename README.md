一、项目描述

- 一个基于 vue、ElementUI，通过 vue 组件实现数据动态刷新渲染，详情具体组件使用情况请点击下方 ElementUI 文档。
- 项目部分区域使用了全局注册方式，增加了打包体积，在实际运用中请使用按需引入。
- 项目环境：vue-cli-3.0、webpack-4.0、npm-6.14.8、node-v14.15.1、element-ui-2.14.1.
